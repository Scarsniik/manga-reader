# Verbe ichidan

Les verbes ichidan gardent un radical stable. On les appelle aussi verbes du groupe 2.

## Formation

On retire généralement le `る` final, puis on ajoute la terminaison.

- `食べる` -> `食べ` + `ない` = `食べない`
- `食べる` -> `食べ` + `た` = `食べた`
- `食べる` -> `食べ` + `て` = `食べて`
- `食べる` -> `食べ` + `よう` = `食べよう`

## Pourquoi c'est utile

Quand tu reconnais un ichidan, la conjugaison est souvent plus prévisible qu'avec un godan.
