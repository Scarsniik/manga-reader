# Verbe suru

Les verbes suru utilisent `する` comme verbe principal ou après un nom verbal. Ils sont irréguliers, mais très fréquents.

## Formation

La base change souvent en `し`.

- Présent affirmatif : `する`
- Présent négatif : `しない`
- Passé affirmatif : `した`
- Passé négatif : `しなかった`
- Forme en て affirmative : `して`
- Volitionnel affirmatif : `しよう`

## Avec un nom

Le nom reste devant, et c'est `する` qui se conjugue.

- `勉強する`
- `勉強しない`
- `勉強した`
- `勉強しました`
