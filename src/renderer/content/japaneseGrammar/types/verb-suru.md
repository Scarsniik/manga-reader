# Verbe suru

Les [verbes](grammar:type:verb) suru utilisent `する` comme [verbe](grammar:type:verb) principal ou après un nom verbal. Ils sont irréguliers, mais très fréquents.

## Formation

La base change souvent en `し`.

- [Présent affirmatif](grammar:form:present-affirmative) : `する`
- [Présent négatif](grammar:form:present-negative) : `しない`
- [Passé affirmatif](grammar:form:past-affirmative) : `した`
- [Passé négatif](grammar:form:past-negative) : `しなかった`
- [Forme en て affirmative](grammar:form:te-form-affirmative) : `して`
- [Volitionnel affirmatif](grammar:form:volitional-affirmative) : `しよう`

## Avec un nom

Le nom reste devant, et c'est `する` qui se conjugue.

- `勉強する`
- `勉強しない`
- `勉強した`
- `勉強しました`

Voir aussi : [passé affirmatif poli](grammar:form:past-affirmative-polite).
