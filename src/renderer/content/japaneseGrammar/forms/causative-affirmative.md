# Causatif affirmatif

Le causatif indique qu'on fait faire une action à quelqu'un (faire faire, laisser faire).

## Comment le former

On transforme le [verbe](grammar:type:verb) selon son groupe :

- [Verbe godan](grammar:type:verb-godan) : changer la finale en ``a`` puis ajouter `せる` : `言う` -> `言わせる`, `行く` -> `行かせる`.
- [Verbe ichidan](grammar:type:verb-ichidan) : retirer `る`, puis ajouter `させる` : `食べる` -> `食べさせる`.
- [Verbe suru](grammar:type:verb-suru) : `する` -> `させる`.
- [Verbe kuru](grammar:type:verb-kuru) : `来る` -> `来させる` (ou `こさせる`).

## À reconnaître

Terminaisons en `せる`, `させる`, `こさせる`. Exemple : `言わせる`, `食べさせる`.
