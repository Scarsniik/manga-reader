# Volitionnel affirmatif

Le volitionnel affirmatif exprime une intention, une proposition ou une décision.

## Comment le former

- [Verbe ichidan](grammar:type:verb-ichidan) : retirer `る`, puis ajouter `よう` : `食べる` -> `食べよう`.
- [Verbe godan](grammar:type:verb-godan) : passer la finale en son `o`, puis ajouter `う` : `読む` -> `読もう`.
- `する` : `しよう`.
- `来る` : `来よう`.

## Sens fréquent

- `行こう` : allons-y, j'y vais.
- `食べよう` : mangeons, je vais manger.
- `見よう` : regardons, je vais regarder.
