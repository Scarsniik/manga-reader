# Causatif — négatif

Le causatif négatif exprime qu'on ne fait pas faire l'action.

## Comment le former

- [Verbe godan](grammar:type:verb-godan) : `言う` -> `言わせない`, `行く` -> `行かせない`.
- [Verbe ichidan](grammar:type:verb-ichidan) : `食べる` -> `食べさせない`.
- [Verbe suru](grammar:type:verb-suru) : `する` -> `させない`.
- [Verbe kuru](grammar:type:verb-kuru) : `来る` -> `来させない`.

## À reconnaître

Terminaisons en `せない`, `させない`. Exemples : `言わせない`, `食べさせない`.
