# Forme potentielle

La forme potentielle exprime la capacité ou la possibilité de réaliser une action.

## Comment la former

On transforme le [verbe](grammar:type:verb) selon son type :

- [Verbe godan](grammar:type:verb-godan) : changer la dernière voyelle en `e`, puis ajouter `る` : `読む` -> `読める`, `行く` -> `行ける`.
- [Verbe ichidan](grammar:type:verb-ichidan) : retirer `る`, puis ajouter `られる` (ou `れる` familier) : `食べる` -> `食べられる` (fam. : `食べれる`).
- [Verbe suru](grammar:type:verb-suru) : `する` -> `できる`.
- [Verbe kuru](grammar:type:verb-kuru) : `来る` -> `来られる` (ou `こられる`).

## À reconnaître

Quand la forme se termine par `める`, `られる` ou `できる`, elle peut être une forme potentielle. Attention : `られる` est souvent ambigu avec le passif ; le contexte est nécessaire pour trancher.
