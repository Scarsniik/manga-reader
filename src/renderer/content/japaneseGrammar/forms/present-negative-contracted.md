# Présent négatif contracté

Le [présent négatif](grammar:form:present-negative) contracté est une version familière de `ない`, très fréquente dans les dialogues. `ない` devient souvent `ねえ` ou `ねぇ`.

## Comment le former

On part de la forme négative neutre, puis on remplace la fin `ない` par `ねえ`.

- [Verbe ichidan](grammar:type:verb-ichidan) : `認めない` -> `認めねえ`.
- [Verbe godan](grammar:type:verb-godan) : `読まない` -> `読まねえ`.
- [Adjectif en い](grammar:type:i-adjective) : `高くない` -> `高くねえ`.
- [Adjectif en な](grammar:type:na-adjective) : `静かじゃない` -> `静かじゃねえ`.

## À reconnaître

Cette forme garde le sens du [présent négatif](grammar:form:present-negative), mais elle sonne orale, relâchée ou rude selon le contexte. Elle n'est pas une forme polie.
