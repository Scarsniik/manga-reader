# Passif affirmatif

Le passif affirmatif indique que le sujet subit l'action ou reçoit l'effet de l'action. En dialogue, une forme comme `やられ` est souvent le radical de `やられる`.

## Comment le former

On met le [verbe](grammar:type:verb) à la base négative, puis on ajoute `れる` ou `られる`.

- [Verbe godan](grammar:type:verb-godan) : `やる` -> `やられる`, `頼る` -> `頼られる`.
- [Verbe ichidan](grammar:type:verb-ichidan) : `食べる` -> `食べられる`, `見る` -> `見られる`.
- [Verbe suru](grammar:type:verb-suru) : `する` -> `される`.
- [Verbe kuru](grammar:type:verb-kuru) : `来る` -> `来られる`.

## À reconnaître

Quand la forme s'arrête sur `れ` ou `られ`, comme `やられ`, elle peut être le radical du passif avant une autre terminaison. Sans contexte plus large, elle reste une forme passive affirmative plutôt qu'un simple [présent affirmatif](grammar:form:present-affirmative).
