# Documentation de CardList.tsx

Ce fichier est un composant React qui sert de conteneur pour afficher une liste de cartes, en l'occurrence des mangas, dans une application de gestion de mangas. Voici une explication détaillée de son fonctionnement :

- Prend en entrée une liste de mangas, et un objet qui défini combien de manga par ligne on affiche suivant les différentes tailles d'écran (responsive).
- Utilise le composant MangaCard pour afficher chaque manga individuellement.
