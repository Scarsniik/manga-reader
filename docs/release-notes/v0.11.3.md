# v0.11.3

## Nouveautes

- Le lecteur OCR peut maintenant lire une bulle avec VOICEVOX depuis le panneau OCR.
- Les options du lecteur ajoutent une section VOICEVOX avec choix de voix, variante, vitesse, hauteur, intonation, volume, pauses et test audio.
- Une option permet de lancer automatiquement la lecture audio de chaque bulle OCR sélectionnée.

## Corrections

- Le bouton de test VOICEVOX conserve les reglages en cours au lieu de remettre les parametres audio aux valeurs par defaut.
