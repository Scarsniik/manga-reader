# v0.8.1

## Nouveautes

- La recherche multi-sources ajoute un filtre texte au-dessus des filtres de langue.
- Ce filtre peut reprendre automatiquement la recherche principale.
- Le filtre accepte plusieurs termes separes par virgule, point-virgule, barre verticale ou retour a la ligne.

## Ameliorations

- Le filtre texte attend un court delai avant de s'appliquer pour garder l'affichage fluide.
- Le bouton de vidage permet de revenir rapidement a tous les resultats charges.

## A savoir

- Ce filtre agit seulement sur les resultats deja charges. Il ne relance pas les scrappers.
