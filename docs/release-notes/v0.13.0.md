# v0.13.0

## Améliorations

- La liste des auteurs extraite depuis une recherche multi-sources est maintenant mémorisée tant que les résultats chargés ne changent pas.
- Quand la récupération des métadonnées depuis les fiches est activée, les auteurs déjà trouvés sont disponibles directement sans charger les fiches une seconde fois.
- La fenêtre des auteurs possède maintenant une recherche locale par auteur, source ou titre.
- L'action `Tout ouvrir` respecte le filtre actif.
- Une étoile jaune identifie les auteurs déjà présents dans les favoris.
