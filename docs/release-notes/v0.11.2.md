# v0.11.2

## Nouveautes

- La vue Nouveautes permet maintenant d'ajuster les principaux parametres de scan uniquement pour la session en cours.
- Le scan des sources peut etre continue depuis l'en-tete sans vider les resultats deja affiches, avec un nombre de passes configurable.
- Le detail des sources indique clairement les sources en cours, celles qui peuvent encore continuer, celles qui n'ont plus de suite et celles en erreur.

## Corrections

- Les sources qui n'ont pas atteint leur quota ne sont plus reprises par le bouton de continuation avance.
- Le bouton de continuation historique sous les resultats est de retour et remplace la liste courante avec la reprise suivante.
