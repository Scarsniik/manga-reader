# v0.5.1

## Nouveautes

- La detection de langue des scrappers peut maintenant utiliser une table de correspondance.
- Dans la section `Langue`, une valeur extraite peut etre associee explicitement a une langue.
- Cette table fonctionne avec le selecteur de langue processed, en CSS comme en regex.
- Si plusieurs valeurs sont trouvees, l'application prend la premiere valeur qui existe dans la table.

## Corrections

- Les regex de langue peuvent lire les informations du conteneur direct d'une card, comme les classes de la `div` principale.
- Les scrappers existants restent compatibles : sans table de correspondance, la detection processed automatique continue de fonctionner.

## A savoir

- La table est utile quand un site renvoie des valeurs non standard, par exemple `flag-eng`, `translated` ou une classe CSS propre au site.
