# v0.8.6

## Ameliorations

- Le panneau OCR permet maintenant de definir l'ordre de lecture des bulles sur une page.
- Les traductions JPDB peuvent utiliser la bulle precedente comme contexte une fois l'ordre valide.
- Deux raccourcis configurables, vides par defaut, permettent de passer a la bulle ordonnee precedente ou suivante.
- L'analyse JPDB des bulles est prechargee pour rendre la lecture OCR plus fluide.

## Corrections

- Le texte affiche dans l'analyse reste limite a la bulle selectionnee apres validation de l'ordre.
- Les numeros d'ordre ne restent visibles que pendant la selection de l'ordre.
- Le contexte JPDB est conserve separement du texte analyse pour eviter de melanger les bulles.
