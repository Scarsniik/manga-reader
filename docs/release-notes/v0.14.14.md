# v0.14.14

## Recherches en arrière-plan

- Les nouveautés des auteurs favoris respectent désormais la limite globale de scraping et le seuil d’arrêt du scan rapide.
- Les pages de détail ne sont plus chargées pour les cartes déjà vues, ce qui accélère fortement les recherches sans nouveau résultat.
- Les paginations répétées sont détectées afin d’éviter les recherches qui ne se terminent pas.

## Correspondances de manga

- Le parseur configuré du scraper est utilisé en priorité, avec le parseur structuré par défaut en solution de repli.
- Les auteurs, titres alternatifs, chapitres, langues et tags de publication sont maintenant séparés correctement.
- Les résultats sont regroupés dans les bons chapitres sans fusionner des mangas différents dans une seule carte.
- Les nouveaux titres principaux découverts alimentent les recherches suivantes, notamment pour retrouver les variantes traduites.
- La recherche brute d’un nom d’auteur est limitée et les tâches auteur redondantes sont éliminées.
