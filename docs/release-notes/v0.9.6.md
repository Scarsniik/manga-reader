# v0.9.6

## Ameliorations

- Les parametres permettent maintenant de regler la limite et la duree de conservation de l'historique des cards vues. Mettre une valeur a `0` rend le reglage illimite.
- La vue `Nouveautes` affiche un avertissement si l'historique des cards vues n'est pas configure en illimite, car d'anciennes cards deja vues peuvent alors reapparaitre apres nettoyage.
- La vue `Nouveautes` attend maintenant un clic sur `Charger` avant de chercher les nouvelles cards.
- L'onglet `Scrappers` des nouveautes peut inclure seulement certaines langues, avec une option `Inconnue` pour garder les cards sans langue detectee. Le choix est conserve pour les prochaines ouvertures.
- Les vues combinees des auteurs et des tags favoris ont maintenant un filtre texte simple sur les cards deja chargees.
- Le detail des tokens japonais indique maintenant le type grammatical et la conjugaison detectee pour les verbes et adjectifs, avec des fiches explicatives navigables.

## Corrections

- Le chargement multipage gere mieux les fins de pagination quand la page suivante repond comme une page inexistante.
- Les auteurs favoris peuvent maintenant mettre leur cache a jour apres un chargement complet, meme quand la derniere page demandee indique simplement qu'il n'y a plus de contenu.
- La recherche multi-sources, les tags favoris et les nouveautes evitent aussi d'afficher une erreur dans ce cas.

## A savoir

- Les vraies erreurs sur la premiere page restent signalees normalement.
