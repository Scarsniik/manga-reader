# v0.11.11

## Corrections

- Les corrections faites dans `Texte analysé` sont maintenant enregistrees en quittant le champ, en plus de `Entrée` et du bouton `Analyser`.
- Le champ indique clairement quand une modification OCR n'est pas encore enregistree.
- Les corrections OCR sont conservees lors d'un recalcul quand la bulle retrouve le meme identifiant.
- La correction OCR peut aussi mettre a jour le cache OCR applicatif quand le fichier OCR du manga n'est pas disponible.
