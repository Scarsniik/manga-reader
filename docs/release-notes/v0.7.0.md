# v0.7.0

## Nouveautes

- Les recommandations de fin de lecture peuvent maintenant inclure les bookmarks scraper.
- L'option se trouve dans les parametres du lecteur.
- L'ecran de fin de lecture ajoute un bouton pour ouvrir un manga aleatoire.

## A savoir

- Les recommandations et le manga aleatoire restent dans la meme langue que la lecture en cours.
- Le manga aleatoire ne choisit pas de chapitre appartenant a une serie.
