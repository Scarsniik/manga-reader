## Listes de lecture depuis une correspondance

- Chaque chapitre absent dans la langue principale peut maintenant être complété avec une source d’une autre langue ou de langue inconnue.
- Le sélecteur indique la langue détectée, le scraper et le titre complet de chaque source proposée.
- La source choisie peut être ouverte dans un nouvel onglet du workspace pour être vérifiée manuellement sans fermer le dialogue.
- Le compteur de la liste se met à jour avec les remplacements sélectionnés et confirme lorsque tous les chapitres ont été complétés.
