# v0.5.0

## Nouveautes

- Les champs de selecteur des scrappers peuvent maintenant etre utilises en mode CSS ou en mode regex.
- Les regex sont appliquees dans le bloc courant, par exemple dans chaque card de resultat ou dans la fiche detaillee.
- Les scrappers peuvent maintenant detecter la langue dans les resultats et les fiches.
- La langue peut etre detectee depuis le titre, depuis une metadata texte, ou depuis une metadata traitee comme les classes de drapeaux de 3hentai.
- Les langues detectees sont affichees avec des drapeaux dans les resultats et les fiches.

## Corrections

- Les regex saisies sous forme `/motif/options` sont maintenant lues correctement dans les champs de configuration.

## A savoir

- Les anciens selecteurs restent compatibles. Les nouveaux champs demarrent en mode CSS par defaut.
