# v0.9.14

## Nouveautes

- Les listes scraper peuvent maintenant enrichir les cards avec leur fiche pendant le chargement. Cette option reste desactivee par defaut pour garder le controle sur la charge reseau.
- La recherche multi-sources dispose de son propre reglage d'enrichissement des fiches, separe des autres listes scraper.
- Depuis une fiche tag, un tag peut maintenant etre blackliste pour le scraper courant.
- Les tags blacklistes sont signales en rouge dans les metadonnees des cards et des fiches.
- Les tags favoris sont signales en rose dans les metadonnees des cards et des fiches.
- Une nouvelle option permet d'exclure completement les cards qui contiennent un tag blackliste des resultats collectes.

## Corrections

- Les tags blacklistes detectes apres enrichment sont maintenant pris en compte dans la selection de la vue Nouveautes.
- La vue Bookmarks conserve son format de cards normal et applique aussi le masquage des tags blacklistes quand les bookmarks contiennent les tags necessaires.
