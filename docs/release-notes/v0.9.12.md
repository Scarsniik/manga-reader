# v0.9.12

## Nouveautes

- Les filtres de recherche multi-sources utilisent maintenant le meme affichage compact que les nouveautes.
- Les filtres de scrappers, langues et types gardent leur configuration entre deux lancements de l'application.
- Un clic droit sur une valeur de filtre permet maintenant de l'exclure. Les valeurs exclues apparaissent en rouge.

## Corrections

- Les recherches multi-sources pre-remplies conservent les filtres deja configures au lieu de les remettre a zero.
