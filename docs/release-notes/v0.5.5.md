# v0.5.5

## Nouveautes

- La vue bookmarks des scrapers ajoute des filtres par texte, langue, nombre de pages et etat de lecture.
- Les bookmarks peuvent etre tries, et un bouton permet de les rescraper pour recuperer les metadonnees manquantes.
- Quand un scraper n'a qu'une langue de source configuree, cette langue est utilisee pour les bookmarks sans langue detectee.

## Corrections

- Le rescrape des bookmarks evite maintenant les ecritures concurrentes qui pouvaient corrompre le fichier de bookmarks.
- Les couvertures des bookmarks passent par le cache de miniatures reduites pour limiter le poids des images affichees.

## A savoir

- Les bookmarks existants restent conserves. Le rescrape enrichit les informations trouvees sans vider la liste.
