# v0.14.4

## OCR

- La reconnaissance des textes japonais est plus rapide et plus fiable, notamment sur les petits textes et certains caractères pleine largeur.
- Le recalcul OCR conserve désormais les corrections manuelles et les sélections déjà effectuées sur la page.
- La file de traitement OCR gère mieux les recalculs simultanés sans écraser les résultats existants.
- Après la mise à jour, utilisez une fois le bouton « Reparer » dans les réglages OCR pour installer le runtime optimisé.
