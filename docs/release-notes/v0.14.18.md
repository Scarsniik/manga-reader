# v0.14.18

## Navigation des correspondances auteur

- Le bouton `Ouvrir l’auteur` mène maintenant correctement à la page auteur correspondante.
- Un clic molette sur ce bouton ouvre la page auteur dans un nouvel onglet du workspace.

## Recherche de correspondance manga

- La recherche par auteur réutilise maintenant la découverte complète des correspondances auteur sur tous les scrapers compatibles.
- Chaque page auteur équivalente trouvée est parcourue pour alimenter les résultats de mangas et de chapitres.
- Les variantes de titres avec un sigle ponctué, comme `NILF` et `N.I.L.F.`, sont désormais reconnues comme identiques.
