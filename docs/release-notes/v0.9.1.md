# v0.9.1

## Nouveautes

- Ajout d'une page Historique accessible depuis le menu principal.
- L'historique regroupe maintenant les lectures, les fiches consultees et les recherches.
- L'historique de lecture permet de reprendre une lecture, supprimer une entree, ajouter un bookmark et marquer une lecture comme lue.
- L'historique des fiches conserve les mangas dont la fiche a ete consultee, avec les actions principales disponibles.
- L'historique de recherche conserve les recherches lancees depuis un scrapper et depuis la recherche multi-source.

## Ameliorations

- La page Historique propose une recherche simple, un filtre par source et une pagination pour mieux gerer les gros historiques.
- Les recherches multi-source affichent les settings utilises au moment du lancement.
- L'onglet des recherches utilise des lignes compactes pour rester lisible avec beaucoup d'entrees.
- Les cards fusionnees indiquent maintenant correctement les nouveautes quand au moins une source n'a pas encore ete vue.
- Les tags favoris peuvent afficher les cards non vues en premier via une option dans les parametres.

## Corrections

- Corrige l'utilisation des URLs placees en variable dans la configuration des scrappers.
- Ces URLs ne sont plus encodees comme des URI, ce qui permet de les reutiliser comme base pour construire d'autres valeurs.
