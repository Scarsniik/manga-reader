# v0.11.7

## Nouveautes

- Le panneau OCR permet maintenant de corriger le texte d'une bulle et d'enregistrer cette correction dans le fichier OCR du manga.
- Le panneau OCR ajoute un bouton de telechargement de l'audio de la bulle courante.
- L'onglet Lecteur permet de choisir le dossier de telechargement audio OCR ; vide, l'application utilise le dossier utilisateur par defaut.

## Corrections

- La lecture et le telechargement audio utilisent le texte OCR corrige, et le cache audio ne reutilise plus un ancien audio quand le texte de la bulle change.
