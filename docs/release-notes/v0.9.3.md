# v0.9.3

## Ameliorations

- Les pages auteur peuvent maintenant basculer entre la vue par pages et une vue combinee.
- La vue combinee reprend le merge, les filtres de langue, les filtres de lecture et les cartes enrichies des auteurs favoris.
- Le choix de vue est disponible dans les settings et directement depuis la page auteur.

## Corrections

- La vue auteur combinee utilise le meme nombre de pages initiales que les auteurs favoris.
- Les actions de page auteur gardent le meme ordre entre la vue combinee et la vue par pages, avec le bouton favori dans les deux vues.
