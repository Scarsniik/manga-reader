# v0.14.1

## Corrections

- Le tri automatique des listes réutilise maintenant les règles personnalisées d'analyse des titres et la normalisation
  des merges de cards pour ignorer correctement les auteurs et les marqueurs de langue.
- Enregistrer une liste existante met désormais à jour cette liste au lieu d'en créer une copie.
- Le survol d'un manga dans une liste affiche maintenant son titre.

## Améliorations

- Le bouton de tri précise lorsque la liste était déjà dans l'ordre détecté.
