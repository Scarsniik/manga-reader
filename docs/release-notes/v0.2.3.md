# v0.2.3

## Nouveautes

- Ajout d'un bouton `Patchnotes` dans `Version et installation` pour lire les 20 dernieres mises a jour sans quitter l'application.
- La fenetre de mise a jour affiche maintenant les patchnotes des versions comprises entre votre version actuelle et la nouvelle version proposee.

## Corrections

- Les patchnotes sont maintenant lues directement depuis les GitHub Releases de l'application.
- La publication d'une release demande desormais une vraie patchnote utilisateur structuree, au lieu d'un texte technique genere automatiquement.

## A savoir

- Les anciennes releases publiees sans patchnote detaillee peuvent encore apparaitre comme non renseignees.
