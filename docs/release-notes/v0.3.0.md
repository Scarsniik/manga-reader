# v0.3.0

## Nouveautes

- Ajout d'un onglet `Raccourcis` dans les parametres.
- Les raccourcis du lecteur peuvent maintenant etre modifies avec une interface de capture au clavier.
- Chaque action peut recevoir jusqu'a trois raccourcis.
- Le lecteur propose un bouton `Parametres` directement dans son header.

## Corrections

- Les raccourcis en doublon sont signales en rouge dans les parametres.
- Modifier un raccourci ne recharge plus la bibliotheque en arriere-plan et ne remet plus la liste des raccourcis en haut.
- Les raccourcis de scroll utilisent de nouveau le bon conteneur dans le lecteur.

## A savoir

- Les raccourcis par defaut du lecteur restent disponibles apres la mise a jour.
