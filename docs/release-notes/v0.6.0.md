# v0.6.0

## Nouveautes

- Ajout des auteurs favoris pour les scrappers.
- Une page auteur peut maintenant etre ajoutee aux favoris avec l'etoile.
- Un auteur favori peut regrouper plusieurs sources venant de scrappers differents.
- La vue `Auteurs favoris` affiche une vue combinee des resultats de toutes les sources rattachees.
- Les resultats combines peuvent etre filtres par langue, comme dans la recherche multi-sources.

## Parametres

- Nouveau reglage global pour choisir combien de pages auteur sont chargees automatiquement a l'ouverture d'un auteur favori.

## A savoir

- Les pages suivantes restent disponibles avec les actions `Charger plus` et `Plus`.
