# v0.8.9

## Nouveautes

- Les auteurs favoris peuvent maintenant stocker leurs resultats pour s'afficher plus vite a la prochaine ouverture.
- Quand le stockage est active, l'application affiche le cache existant tout de suite puis re-scrape l'auteur en arriere-plan pour mettre les resultats a jour.
- La page d'un auteur favori propose maintenant `Charger tout` pour charger toutes les pages restantes en une seule action.
- Les sources d'un auteur favori sont cliquables et ouvrent directement la page auteur dans le scrapper correspondant.
- Depuis une page auteur deja rattachee a un auteur favori, un bouton permet d'ouvrir directement la page favori de cet auteur.
- Dans les resultats fusionnes, cliquer sur le bloc `En cours` ouvre maintenant le lecteur directement a la page sauvegardee.
- Les cards fusionnees ont maintenant un bouton compact pour marquer le resultat comme lu ou non lu depuis le bloc de progression.

## Corrections

- Dans le reader, les raccourcis OCR peuvent maintenant sélectionner le premier résultat même si aucune bulle n'était déjà sélectionnée.
- Le recentrage automatique sur une bulle OCR ne bloque plus le scroll manuel du reader.
- La pagination des pages auteur fonctionne mieux avec les scrappers qui utilisent un template d'URL, notamment pour atteindre la deuxieme page d'un auteur.
- Les resultats fusionnes ne comptent plus deux fois la meme source quand le cache et le rafraichissement en arriere-plan retournent la meme URL.
- Les badges comme `Bookmark x2` ne s'affichent plus quand les doublons pointent en fait vers le meme manga.
- Les options des reglages desactivees sont maintenant visuellement grisees.
- Les filtres de lecture sur les resultats fusionnes masquent ou affichent maintenant la card entiere, sans retirer seulement une source du merge.
- Le marquage `Lu` cible la bonne source : la source en cours quand elle existe, sinon seulement la premiere source d'un resultat pas encore commence.
- Le menu `Ouvrir avec` aligne mieux la langue, le nom de la source, la progression et le bookmark, et le bookmark retrouve son badge jaune avec une icone.

## A savoir

- Activer le stockage des auteurs favoris implique de charger toutes les pages disponibles de l'auteur pour construire le cache.
- Les caches sont stockes dans le dossier de donnees utilisateur, dans `scraper-author-favorite-cache`.
