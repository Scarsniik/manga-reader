# v0.3.2

## Nouveautés

### Recherche multiple

- Possibilité de faire une recherche multiple sur plusieurs termes (ex: "my hero academia, boku no hero academia, mha" pour faire une recherche sur un manga avec plusieurs titres)
- Affichage d'une icone sur les résultats de recherche multiple qui sont déjà dans la bibliothèque ou en bookmark
- Affichage de la liste des scrappers dans une card qui s'ouvre en hover pour éviter de déplacer les autres résultats de recherche
- Lazy loading des résultats de recherche multiple (ça rame toujours sur les grosses recherches mais c'est mieux)
- Drapeau pour les langues dans les résultats de recherche multiple

## Fixes

- Meilleurs merge dans la recherche multiple. Moins de doublons et de résultats qui n'apparaissent pas
