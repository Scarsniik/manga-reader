# v0.8.8

## Nouveautes

- La recherche multi-sources affiche maintenant la progression de lecture directement sur les cards fusionnees.
- Le menu `Ouvrir avec` indique les sources deja en bibliotheque, en bookmark ou avec une progression en cours/terminee.
- Les resultats multi-sources et les auteurs favoris peuvent etre filtres par etat de lecture : `Non lu`, `En cours` et `Lu`.

## Corrections

- Les changements de progression du lecteur scraper rafraichissent maintenant les vues qui affichent ces etats.
