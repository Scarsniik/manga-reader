# v0.9.10

## Nouveautés

- Les scrappers peuvent maintenant configurer une analyse des titres pour isoler le titre principal, les auteurs, le cercle, la parodie, la langue, les tags de suffixe et les indicateurs de chapitre, partie ou tome.
- La configuration de cette analyse se teste depuis des titres saisis manuellement ou depuis des résultats chargés par le module Recherche du scrapper.
- Les fiches manga proposent une recherche multi-sources préremplie avec le titre principal détecté, ouverte dans la vue courante ou dans un nouvel onglet workspace au clic molette.
- Les blocs optionnels de l'analyse de titre peuvent maintenant être ignorés automatiquement si leur consommation empêche la suite de la variante de correspondre.

## Corrections

- Les patchnotes affichées dans la fenêtre de mise à jour réparent maintenant les anciens textes publiés avec un mauvais encodage.
- Les accents, les caractères japonais et les autres caractères non ASCII s'affichent correctement dans les patchnotes.
- On peut copier le texte tokenisé de l'ocr normalement désormais.

## À savoir

- La publication des prochaines releases lit aussi les patchnotes en UTF-8 pour éviter que le problème revienne.
