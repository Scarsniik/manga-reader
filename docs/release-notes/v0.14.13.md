# v0.14.13

## Recherches de nouveautés en arrière-plan

- Une source qui renvoie plusieurs fois la même page est désormais arrêtée automatiquement.
- La recherche interrompt la compensation après plusieurs pages sans aucun nouveau résultat à examiner.
- Une même carte blacklistée n’est plus comptée plusieurs fois lorsqu’une source répète ses résultats.
