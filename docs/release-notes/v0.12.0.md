# v0.12.0

## Nouveautés

- Les listes de lecture peuvent maintenant être enregistrées et retrouvées après le redémarrage de l'application.
- La page Bookmarks possède un nouvel onglet `Listes` avec le nombre de mangas, un aperçu des couvertures et la date d'enregistrement.
- Chaque liste enregistrée peut être ouverte pour être préparée, lancée directement en lecture ou supprimée.
- Dans le lecteur, un bouton permet de passer au manga suivant de la liste sans marquer le manga courant comme lu.

## Améliorations

- L'option qui retire le bookmark après lecture est maintenant activée par défaut pour les listes de lecture.
