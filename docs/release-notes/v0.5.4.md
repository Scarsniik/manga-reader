# v0.5.4

## Nouveautes

- La recherche multi-sources permet maintenant de filtrer les resultats visibles par langue directement depuis les drapeaux sous le compteur.
- Le clic gauche sur un drapeau affiche seulement cette langue. Le clic droit masque cette langue.

## A savoir

- Le filtre ne supprime pas les resultats charges. En retirant le filtre, les resultats reviennent comme avant.
