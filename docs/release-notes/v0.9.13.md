# v0.9.13

## Nouveautes

- Le nombre de chargements simultanes du mode Nouveautes est maintenant configurable dans les Options. La valeur minimum est 1.
- Les fiches scraper signalent maintenant les mangas potentiellement deja lus, en cours ou deja bookmarkes.
- Depuis ce signalement, un bouton permet d'ouvrir la fiche trouvee ou de retrouver le manga local dans la bibliotheque avec une recherche exacte.
- La vue Nouveautes des sources n'applique plus de fusion entre resultats : chaque resultat reste visible separement.
- Les resultats Nouveautes des sources sont affiches du plus vieux au plus recent pour chaque scraper.
- Les limites de resultats Nouveautes sont maintenant separees entre les scrapers et les tags favoris.
- Les Options ajoutent une section Scraping qui regroupe les reglages des scrapers, des nouveautes et de l'historique de cards vues.

## Corrections

- Le scan Nouveautes peut maintenant arreter automatiquement une source qui accumule trop de resultats refuses par langue sans jamais trouver de resultat dans une langue demandee.
- Si une source trouve au moins un resultat dans une langue demandee, les refus de langue suivants restent seulement informatifs et ne coupent plus le scan.
