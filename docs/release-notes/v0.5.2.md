- La fenetre workspace se rouvre maintenant a l'endroit ou elle avait ete placee la derniere fois.
- Sa taille est aussi conservee apres un redimensionnement.
- Dans la configuration `Auteur`, la detection de langue peut etre copiee depuis `Recherche`.
- La copie reprend aussi la table de correspondance utilisee par la detection processed.

- Si l'ecran utilise precedemment n'est plus disponible, la fenetre revient sur un placement utilisable au lieu de se retrouver hors ecran.
