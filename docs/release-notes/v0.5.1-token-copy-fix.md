# v0.5.1

## Nouveautes

- Amélioration du comportement de copie dans l'analyseur japonais : lorsqu'on copie la phrase tokenisée, le texte collé est désormais la phrase normale (sans retours à la ligne entre les tokens). Fichier modifié : `src/renderer/components/JapaneseAnalyse/TokensList.tsx`.

## Corrections

- Correction d'un bug où la copie insérait un retour à la ligne entre chaque token.

## A savoir

- La correction intercepte l'événement `onCopy` et colle le texte brut de la phrase. Si vous préférez une correction côté CSS, voir `src/renderer/components/JapaneseAnalyse/TokensList.scss`.
