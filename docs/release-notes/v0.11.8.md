## Nouveautes

- Le detail d'un token OCR ajoute un bouton lecture pour jouer uniquement ce token avec les reglages VOICEVOX courants.

## Corrections

- La correction du texte d'une bulle OCR se fait maintenant uniquement depuis la zone `Texte analysé`, sans deuxieme zone d'edition dans le panneau OCR.
- La validation de `Texte analysé` persiste toujours la correction dans la bulle OCR selectionnee avant de relancer l'analyse.
