# v0.14.0

## Nouveautés

- Les mangas d'une liste de lecture peuvent maintenant être réorganisés par glisser-déposer avant de lancer la lecture.
- Un bouton de tri automatique détecte les numéros de tome ou de chapitre dans les titres similaires pour les remettre dans l'ordre.

## Améliorations

- La poignée de déplacement permet aussi de réorganiser la liste avec les flèches du clavier.
