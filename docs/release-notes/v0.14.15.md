# v0.14.15

## Cartes fusionnées

- Les paramètres globaux permettent maintenant d’ajouter plusieurs langues prioritaires pour le titre des cartes.
- Les langues peuvent être réordonnées par glisser-déposer et retirées individuellement.
- Le titre affiché vient de la première source trouvée dans la langue disponible la plus prioritaire.

## Correspondances de manga

- La vue par chapitre affiche désormais une seule carte par chapitre avec toutes les sources correspondantes.
- Les cartes ne reçoivent plus toutes le même titre artificiel : elles utilisent le véritable titre de la source prioritaire.
- Le parseur reconnaît les préfixes de convention, les métadonnées imbriquées, les chapitres ponctués et les numéros placés avant plusieurs blocs entre parenthèses.
- Les résultats temporaires déjà enregistrés sont réanalysés à l’affichage afin de récupérer les chapitres auparavant non renseignés.
