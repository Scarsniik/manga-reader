# v0.9.19

## Nouveautes

- Les bookmarks ont maintenant une vue "Tags frequents" pour voir les tags les plus presents, filtrer rapidement sur un tag et ouvrir le resultat dans un onglet workspace avec le clic molette.
- La vue des tags frequents permet de choisir le perimetre, le nombre minimum d'occurrences et une fusion fuzzy pour regrouper les variantes proches d'un meme tag.
- Les scrappers peuvent maintenant alimenter leur cache de tags automatiquement depuis les fiches ouvertes, sans page de liste de tags dediee.

## Corrections

- Le cache de tags fusionne mieux les doublons par nom ou par lien et conserve les tags deja connus quand de nouveaux tags sont collectes depuis les fiches.
- Les onglets workspace ouverts depuis les bookmarks conservent mieux les filtres actifs.
- Les resultats de nouveautes regroupent de nouveau les cards equivalentes comme la recherche multi-sources.
