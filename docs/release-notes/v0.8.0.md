# v0.8.0

## Nouveautes

- Les scrappers peuvent maintenant avoir un module `Homepage`.
- La homepage charge une page fixe sans terme de recherche et affiche ses cards comme une recherche.
- La configuration de la homepage peut recopier les selecteurs et la detection de langue depuis `Recherche`.

## Ameliorations

- L'ouverture d'un scrapper utilise la homepage comme accueil quand ce module est configure.
- La pagination de homepage fonctionne avec les templates de page ou avec un lien HTML de page suivante.

## A savoir

- L'ancienne recherche d'accueil reste disponible dans les reglages globaux pour les scrappers qui n'utilisent pas encore le module `Homepage`.
