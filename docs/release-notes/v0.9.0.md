# v0.9.0

## Nouveautes

- Ajout de la gestion des pages tag dans les scrappers.
- Les tags peuvent maintenant etre ouverts depuis une fiche quand le scrapper expose leurs liens.
- Ajout des tags favoris, avec rattachement de plusieurs sources tag a un meme favori.
- La page d'un tag favori affiche les resultats combines des sources associees, avec pagination globale.
- Les pages de tags favoris affichent maintenant la liste des sources et un filtre par langue.

## Ameliorations

- Le formulaire d'ajout aux favoris propose automatiquement l'auteur ou le tag existant le plus proche du nom courant.
- La configuration auteur et tag partage davantage de logique interne, ce qui reduit la duplication dans l'editeur de scrappers.

## Corrections

- Corrige la navigation depuis une fiche ouverte via un tag : cliquer sur un autre tag ouvre bien ce nouveau tag au lieu de revenir au precedent.
