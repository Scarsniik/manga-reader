# v0.8.10

## Nouveautes

- Une recherche multi-sources peut maintenant etre arretee globalement.
- Une recherche multi-sources peut aussi etre arretee pour un scrapper precis depuis le detail des statuts.
- La recherche multi-sources peut maintenant extraire la liste des auteurs depuis les resultats charges.
- La liste des auteurs trouves indique leur nom et leur scrapper, avec une action pour ouvrir chaque page auteur ou tout ouvrir d'un coup.
- Les auteurs deja presents dans un favori auteur sont signales directement dans la liste des auteurs trouves.
- La configuration d'une page auteur accepte maintenant un selecteur dedie pour recuperer le nom de l'auteur.
- Quand le nom est disponible, la page auteur affiche ce nom a la place de `Resultats auteur`, avec une majuscule au debut de chaque mot.
- Depuis une page auteur ou un auteur favori, un bouton permet d'ouvrir la recherche multi-sources avec les noms auteur deja pre-remplis, sans lancer la recherche automatiquement.

## A savoir

- Arreter une recherche ignore les resultats qui arrivent ensuite, mais les requetes deja envoyees peuvent encore se terminer cote scrapper.
- Le pre-remplissage depuis une page auteur dedoublonne les noms connus et les separe par `, `.
