# v0.9.9

## Ameliorations

- Les fiches explicatives de grammaire japonaise integrent maintenant leurs liens directement dans le texte pour faciliter la navigation entre les notions liees.
- Dans les onglets workspace, les fiches peuvent ouvrir le lecteur sans masquer la barre d'onglets. Les tags peuvent aussi s'ouvrir dans un nouvel onglet avec le clic molette.
- La vue `Nouveautes` peut maintenant scanner les tags favoris en plus des scrappers. Les scrappers ont aussi une option `Aucun`, ce qui permet de lancer les nouveautes uniquement sur les tags favoris. Le scan reste bloque si aucune source n'est selectionnee.
- L'onglet `Auteurs` de la vue `Nouveautes` permet maintenant de choisir les auteurs favoris inclus, avec les options `Tous` et `Aucun`.

## Corrections

- La progression affichee sur les cartes de serie de la bibliotheque se met maintenant a jour des le retour depuis le lecteur, sans attendre un redemarrage.
- Le bouton retour du lecteur dans un onglet workspace revient a la fiche d'origine quand elle est dans le meme onglet, et n'apparait pas quand le lecteur a ete ouvert dans un onglet separe.
- Les dialogues d'aide ouverts depuis l'OCR du lecteur s'affichent maintenant aussi en plein ecran.
